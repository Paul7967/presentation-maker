**Версия:** 1.0  
**Этап:** 5 — Тестирование  
**Примечание:** В проекте не настроены Playwright или Cypress. Ниже — пошаговые сценарии для ручной проверки.

Связь: [test-plan.md](test-plan.md), [design-spec.md](design-spec.md), [api.md](api.md).

## 1. Успешный сценарий: ввод → генерация → скачивание

| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть приложение | Отображается форма: блок «Описание слайдов», блок «Файл-образец (PPTX)», кнопка «Сгенерировать» (неактивна при пустой форме). |
| 2 | Ввести заголовок первого слайда и один пункт | Поля заполнены, отображается счётчик символов. |
| 3 | При необходимости нажать «Добавить слайд» и заполнить второй слайд | Появляется карточка «Слайд 2», кнопка «Удалить слайд» доступна. |
| 4 | В блоке «Файл-образец» выбрать файл .pptx (до 20 MB) | Отображается имя файла и размер. |
| 5 | Нажать «Сгенерировать» | Появляется индикатор загрузки («Генерация презентации…»), кнопка недоступна. |
| 6 | Дождаться ответа сервера (200) | Состояние «Успех»: сообщение об успехе, кнопки «Скачать» и «Создать ещё». |
| 7 | Нажать «Скачать» | Файл presentation.pptx сохраняется и открывается в приложении, поддерживающем PPTX. |
| 8 | Нажать «Создать ещё» | Возврат к форме (очищенной). |

## 2. Граничные случаи

### 2.1 Пустой ввод

- Не вводить контент слайдов: кнопка «Сгенерировать» неактивна или при отправке — «Введите описание хотя бы одного слайда».

### 2.2 Неверный формат файла

- Выбрать файл не .pptx: ошибка «Поддерживается только формат PPTX», файл не принимается.

### 2.3 Размер файла > 20 MB

- Выбрать .pptx > 20 MB: ошибка «Превышен максимальный размер файла (20 MB)».

### 2.4 Более 100 слайдов

- Добавить слайды до лимита: кнопка «Добавить слайд» недоступна при 100 слайдах; при отправке с >100 — сообщение о лимите.

### 2.5 Ошибка API / сеть

- Остановить backend, нажать «Сгенерировать»: состояние «Ошибка», кнопка «Повторить». После запуска backend «Повторить» отправляет запрос снова.

## 3. Состояния UI

- **Форма:** поля редактируемые, подсказки про лимиты видны.
- **Загрузка:** спиннер, текст «Генерация презентации…», форма не активна.
- **Успех:** сообщение об успехе, «Скачать», «Создать ещё».
- **Ошибка:** текст ошибки (detail), кнопка «Повторить», форма доступна для правок.
