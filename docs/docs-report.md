## Выполненные пункты

### 7.1 README проекта

- **Назначение приложения:** в корневом README уже было указано — веб-приложение для создания презентаций PPTX из текстового описания слайдов и файла-образца.
- **Запуск (frontend + backend):** инструкции по запуску backend (venv, pip, uvicorn с портом 8000 и таймаутом 120) и frontend (npm install, npm run dev, порт 5173) сохранены и актуальны.
- **Переменные окружения:** добавлен раздел «Переменные окружения»:
  - **Frontend:** переменная `VITE_API_URL` (описание, значение по умолчанию `http://localhost:8000`, пример для production).
  - **Backend:** указано, что для локальной разработки переменные не требуются; для деплоя — рекомендации по лимиту тела (20 MB), таймауту (120 с) и CORS (ограничение allow_origins в production).
