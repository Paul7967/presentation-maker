## Выполненные обновления

### 8.1. docs/api.md

- Обновлено на этапе 2 (architect): в таблицу лимитов добавлена максимальная длина заметок (4000 символов); в формат поля `slides` добавлены опциональные поля **layoutId** и **notes** с описанием и примером; в раздел валидации добавлена проверка layoutId и notes; в таблицу кодов ошибок уточнено сообщение при нарушении лимитов (в т.ч. заметок).

### 8.2. README.md

- **Режим «По слайдам»:** добавлено описание возможности задавать тип шаблона (layoutId) и заметки докладчика для каждого слайда, указан лимит 4000 символов для заметок.
- **Режим «Через JSON»:** в описание формата добавлены поля `layoutId` и `notes`.
- **Лимиты:** добавлено упоминание лимита заметок (до 4000 символов на слайд).

### 8.3. docs/architecture.md

- В раздел «4.2 Работа с PPTX» обновлён пункт о генерации: первый слайд перезаписывается контентом и заметками; для следующих слайдов выбор layout по layoutId (индекс или имя, fallback — первый layout), запись заметок в slide.notes_slide.
